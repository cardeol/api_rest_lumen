A JSON Restful API Boilerplate written in PHP/Lumen/Laravel with JWT Auth

## Commands

```
composer install 
php artisan migrate
php artisan db:seed
```

## Running tests

```
vendor/bin/phpunit
```

## Local Server

```
php -S localhost:8000 -t public
```
